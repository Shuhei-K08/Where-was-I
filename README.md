# Where-was-I
"Where-was-I"は顧客情報や商談内容のメモなどを管理するアプリケーションです。 <br>
 商談メモ欄に誰といつどのような商談をしたかなど細かく管理することができます。<br>
 また、Todoリスト機能もあり商談メモをとりながらTodoリストの登録もでき、<br>
 作業の抜け漏れを防止し、やるべきことを適切に管理することができます。

- 企業名の登録、閲覧
- 担当者の登録、閲覧
- 商談メモの作成、閲覧
- TodoListの追加、削除
